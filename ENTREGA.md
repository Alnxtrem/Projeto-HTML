## 4. Código-fonte

Cole o código HTML, CSS e JavaScript no PDF somente se você não conseguir publicar o site corretamente na internet.

## Checklist final

- O site está publicado e acessível pela internet.
- O repositório está público no GitHub.
- O endereço publicado contém o nome do aluno.
- Os prints mostram a URL visível no navegador.
- O PDF foi salvo no formato `.pdf`.
- A entrega será enviada no AVA, na seção `Trabalhos`.
- Conferi tudo antes de enviar, porque não será possível reenviar ou substituir arquivos depois.
